# input_helper

A small tool that allows you to achieve the paste effect by simulating input on web pages/programs where pasting is not allowed.

**You need to operate using an English input method.**

一个帮助你在不能粘贴的程序里通过模拟键盘输入来达到类似粘贴的效果。

**你需要在英文输入法下进行操作。**

```bash
pip install pynput
python input_helper.py
```

or:

download input_helper.exe/input_helper.app at Release.

或者：

在Release中下载exe或app文件。

If you are using macOS, please grant the necessary permissions in the prompt window the first time you use the tool.
